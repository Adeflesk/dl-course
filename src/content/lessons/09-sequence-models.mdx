## Overview

Sequence models process variable-length inputs by maintaining hidden state across timesteps.

### Problem: Vanishing Gradients

RNNs struggle with long sequences because gradients shrink exponentially during backprop through time (BPTT).

### LSTM Solution

LSTMs add gating mechanisms and cell state to preserve gradients:

- **Forget gate** — controls which information to discard
- **Input gate** — controls what new information to add
- **Output gate** — controls what to expose to next layer
- **Cell state** — long-term memory that flows through timesteps

The additive cell state update creates skip connections for gradient flow.
